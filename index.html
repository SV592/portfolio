Exploring Dependency Related Build Breakages In The NPM Ecosystem

This research aims to examine the prevalence and implications of dependency-related build failures in software development, specifically in the NPM ecosystem. It seeks to identify the main causes of such failures and their impact on project stability and reliability.

Code Review Practises On Ethereum Smart Contracts

The aim of this is to evaluate the code review process for Smart Contracts. It aims to assess the accuracy of pull request reviews, identify potential vulnerabilities or bugs introduced through merged requests of varying levels of project maturity.

Predicting Build Breakage With Machine Learning.

This research aims to provide a comprehensive overview of existing research on predicting build breakages using machine learning techniques. By examining ml models pros, cons, data sources and evaluation metrics, etc.

Exploring the Prevalence of Social Biases In State Of The Art large language Models

This research aims to investigate social biases within state-of-the-art large language models, focusing on gender, ethnicity, age, etc. It seeks to address biases and promote inclusive AI systems, providing valuable resources for future studies.

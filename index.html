Papers

Analyzing Dependency Related Build Breakage In The NPM Ecosystem

The paper discusses the critical importance of dependency management in software development to ensure projects build and run smoothly. It highlights the risks associated with dependency updates that may cause build breakages, leading to delays in development and potential loss of trust in the software.

Code Review Practises On Etherum Smart Contracts

The paper discusses the importance of version control systems like Git in software development, particularly in the context of code review and integration. It highlights the use of branches and forks in Git to facilitate the seamless merging of code changes into the central repository of smart contracts using solidity.

Predicting Build Breakage With Machine Learning.

The paper aims to provide a comprehensive overview of existing research on predicting build breakages using machine learning techniques. It examines various machine learning models pros, cons, data sources and evaluation metrics, etc while investigating the impact of factors such as code complexity and code change frequency etc.

Exploring the Prevalence of Social Biases In State Of The Art large language Models

The paper investigates social biases present in state-of-the-art large language models, which have shown remarkable success in natural language processing tasks. It aims to quantify biases in specific domains such as gender, ethnicity, and age, conducting a systematic analysis to identify instances of biased responses.
